# Product Overview

## Student Portal Management System

A comprehensive web-based management system for educational institutions with role-based access for students, teachers, and administrators.

## Core Purpose

Streamline academic administration, improve communication between stakeholders, and provide 24/7 access to academic information.

## User Roles

- **Students**: View marks/grades (GP/CP/GPA/CGPA), check attendance, submit assignments, pay fees, download receipts, view notices, access virtual ID card
- **Teachers**: Manage student lists, mark attendance, enter marks/grades, create and track assignments, view student performance
- **Admins**: User management, fee structure management, payment processing, send notifications, generate reports

## Key Features

- Real-time grade calculations (GP/CP/SCPA)
- PDF receipt generation
- Role-based dashboards
- Responsive design (desktop, tablet, mobile)
- Glassmorphism UI with dark mode
- Three-tier fee deadline system (no fine, fine, super fine)

## Academic Structure

- 6 semesters (3 years)
- Odd semesters have higher fees than even semesters
- Subjects managed by admin in database
- Fixed grading scale (hardcoded, not configurable)

## Important Constraints

- Mock payment system (no real payment gateway)
- Admin-only password reset (no email reset)
- Single super admin account
- Marks locked after semester ends
- Fee notices sent after enrollment closes
- No partial fee payments
